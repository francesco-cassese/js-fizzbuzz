# JS FizzBuzz

## 📌 Descrizione
Esercizio classico di logica di programmazione, focalizzato sulla gestione dei cicli e sulla verifica delle condizioni di divisibilità tra numeri. L'obiettivo è generare una sequenza numerica con sostituzioni testuali specifiche.

## 🎯 Obiettivo
L'obiettivo principale è allenare il pensiero algoritmico, imparando a gestire la precedenza delle condizioni logiche all'interno di un'iterazione.

### Focus Tecnico:
- **Cicli (Loops):** Utilizzo del ciclo `for` per generare una progressione numerica da 1 a 100.
- **Operatore Modulo (%):** Utilizzo dell'operatore `%` per determinare se un numero è divisibile per un altro (verificando se il resto è zero).
- **Gerarchia Condizionale:** Organizzazione delle istruzioni `if`, `else if` ed `else` per intercettare correttamente i multipli comuni prima di quelli singoli.

## 🛠️ Tecnologie Utilizzate
- **HTML5:** Struttura base per l'integrazione del file script.
- **JavaScript (ES6+):** Sviluppo della logica di controllo e output in console.

---

## 🧪 Casi di Test
Per verificare la correttezza della logica:
- Multipli di 3 (es. 3, 6, 9) => **Fizz**
- Multipli di 5 (es. 5, 10, 20) => **Buzz**
- Multipli di entrambi (es. 15, 30, 45) => **FizzBuzz**
- Altri numeri (es. 1, 2, 4) => **Valore numerico**
